: Accounts report

Company's client data in tabular form. The first column specifies a group, the second one contains the total number of accounts followed by the percentage of active clients (trading activity within the last three months detected), fixed profit, floating profit or loss, equity and the group's currency. The summary data grouped by currency is displayed at the bottom.

: Deals Count Report

Amount of performed deals. Deal types are highlighted in different colors.

: Deals Volume Report

Volumes of performed deals in one of the currencies (USD is used by default, customizable). Deal types are highlighted in different colors.

: Deals Report (monthly)

Tabular report for each deal type. Each line contains the appropriate monthly data. Each cell contains two values, count of deals and performed volume in one of the currencies (USD is used by default, customizable). Summary data for the entire time interval is displayed at the bottom.
